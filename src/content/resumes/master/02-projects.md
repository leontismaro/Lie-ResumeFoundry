### 企业级 AI 知识助手平台

面向内部知识管理与运营协作场景构建的多租户问答与检索平台。

- 负责文档解析、分片、向量化存储、权限隔离与对话上下文管理等核心模块设计。
- 实现统一模型接入层与 RAG 检索链路，支持多数据源接入、引用追踪与运营审计。
- 为平台补充评测与反馈机制，帮助业务团队持续优化问答质量与召回表现。

### 多租户 API Gateway 与计费系统

面向多产品线统一接入外部能力与内部服务的网关系统。

- 设计统一认证、限流、配额、计费与审计能力，减少各业务线重复接入成本。
- 支持多供应商切换、故障降级与流量路由策略，实现更稳定的外部依赖管理。
- 为运营与财务侧提供成本明细、用量统计与异常告警视图。

### 运营自动化控制台

面向业务运营团队的多模块后台系统，用于配置规则、查看指标、处理异常与执行高频操作。

- 负责复杂表单、数据表格、审计视图和权限逻辑的全栈实现。
- 将分散在多个工具中的操作流程收口到统一工作台，减少重复跳转和人工校对。
- 通过埋点和使用反馈迭代关键页面，持续优化高频路径的完成效率。

### 协作型项目管理平台

面向跨职能团队的任务协作与项目跟踪系统示例。

- 使用 React、TypeScript 与 Node.js 构建前后台一体化应用，覆盖任务流、评论、标签与权限管理。
- 负责数据模型设计、API 约束定义与前端状态管理方案，保证多人协作场景下的交互一致性。
- 将复杂业务页面拆分为可复用模块，降低后续功能扩展成本。
